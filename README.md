# My Portfolio Site


## Project Setup
~~~~
```sh
npm install
```

### Compile and Hot-Reload for Development
~~~~
```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/) and [Stylelint](https://stylelint.io/)

```sh
npm run lint
```
